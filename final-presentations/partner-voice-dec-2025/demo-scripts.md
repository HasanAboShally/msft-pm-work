## Demo 1: Fabric CLI Basics (8 min)

### Narrative
> "Let me show you how the Fabric CLI lets you treat Fabric like a filesystem. If you've ever used `ls`, `cd`, or `cp` in a terminal, you already know the mental model."

### Commands

```bash
# 1. Check authentication (show you're connected)
fab auth status

# 2. List all workspaces (like 'ls' at root)
fab ls

# 3. List contents of a specific workspace with details
fab ls "Demo-Partner-Voice.Workspace" -l

# 4. Get detailed info about an item
fab get "Demo-Partner-Voice.Workspace/SalesData.Lakehouse"

# 5. Create a new workspace (like mkdir)
fab mkdir "Demo-New-Workspace.Workspace"

# 6. Verify it was created
fab ls "Demo-*.Workspace"

# 7. Copy an item between workspaces (like cp)
fab cp "Demo-Development.Workspace/SampleReport.Report" "Demo-Production.Workspace" -f

# 8. Run a notebook job
fab job run "Demo-Partner-Voice.Workspace/ETL-Process.Notebook"
```

## Demo 3: AI Agent + CLI Integration (10 min)

### Narrative
> "Now let's see the real magic - what happens when we combine Copilot's natural language understanding with CLI's precision. Imagine you need to rename 50 workspaces to match a naming convention. Doing this manually would take hours."

### Scenario Setup
Explain the problem:
> "Our company wants all workspaces to follow the pattern: [Environment]-[Domain]-[Project]. For example: Prod-Sales-Dashboard."

### Demo Script

**Step 1: Natural Language Request**
```
User: "I want all my workspaces to follow this naming pattern: 
       [Environment]-[Domain]-[Project]
       For example: Prod-Sales-Dashboard
       
       Can you analyze my current workspaces and propose new names?"
```

**Step 2: AI Analyzes and Proposes (Show the AI's response)**
```
AI Response:
"I've analyzed your workspaces. Here's my proposal:

| Current Name | Proposed Name |
|--------------|---------------|
| Sales-Prod | Prod-Sales-Main |
| Dev-Analytics | Dev-Analytics-Main |
| Finance_Reports | Prod-Finance-Reports |

I can generate a CLI script to rename these. Would you like me to proceed?"
```

**Step 3: AI Generates CLI Script**
```bash
# Generated by AI Agent
#!/bin/bash
# Workspace Rename Script - Generated 2025-12-02

fab set "Sales-Prod.Workspace" -q displayName -i "Prod-Sales-Main"
fab set "Dev-Analytics.Workspace" -q displayName -i "Dev-Analytics-Main"  
fab set "Finance_Reports.Workspace" -q displayName -i "Prod-Finance-Reports"

echo "Rename complete!"
```

**Step 4: User Reviews and Approves**
> "The user reviews the script - they can see exactly what will happen. They can modify it if needed. Then they execute."

**Step 5: Execute (or show what would happen)**
```bash
# Run the script
./rename-workspaces.sh

# Or run directly:
fab set "Demo-Development.Workspace" -q displayName -i "Dev-Demo-Development"
```
