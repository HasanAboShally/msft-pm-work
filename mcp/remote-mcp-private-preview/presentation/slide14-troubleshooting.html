Troubleshooting

"Invalid workspace ID"

IDs must be valid UUIDs

Solution

Use list_workspaces or list_items first to get correct IDs

Authentication Failed (401/403)

Token expired or wrong scope

Solution

Disconnect and reconnect MCP server in VS Code. For manual tokens: az account get-access-token --resource https://api.fabric.microsoft.com

"Workspace not found"

Exists but you can't access it

Solution

Ask workspace Admin to grant you at least Viewer access

Long-running operation timeout

Some ops are async

Solution

Use get_operation_state with operationId, then get_operation_result once it succeeds
