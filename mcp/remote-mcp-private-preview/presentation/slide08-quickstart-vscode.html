Quick Start: VS Code + GitHub Copilot

Recommended setup with interactive authentication

1

Open MCP Server Configuration

Press Cmd+Shift+P (macOS) or Ctrl+Shift+P (Windows)

Type "MCP: Add Server" and select HTTP

2

Enter the Fabric MCP Server URL

https://api.fabric.microsoft.com/v1/mcp

Name it "fabric" when prompted

3

Authenticate

VS Code will open your browser for sign-in. Use your Entra ID account.

4

Verify Connection

Ask Copilot: "List all my Fabric workspaces"

Manual Config Alternative

Add to .vscode/mcp.json:

{
  "servers": {
    "fabric": {
      "type": "http",
      "url": "https://api.fabric
        .microsoft.com/v1/mcp"
    }
  }
}
