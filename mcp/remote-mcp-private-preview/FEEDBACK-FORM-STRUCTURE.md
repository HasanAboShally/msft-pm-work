# Fabric MCP Server - Private Preview Feedback

**Form Link:** https://aka.ms/FabricMCP/PrivatePreview/SubmitFeedback

---

## Questions

**1. Name / Email / Company**
- Type: Short text
- Required: Yes
- Placeholder: "Jane Doe, jane@contoso.com, Contoso"

**2. Agent**
- Type: Choice
- Required: Yes
- Options: VS Code + Copilot | Claude Desktop | Other

**3. Title**
- Type: Short text
- Required: Yes
- Placeholder: "Could not create workspace"

**4. Describe the issue**
- Type: Long text
- Required: Yes
- Hint: "Steps, what happened, what you expected"

**5. Severity**
- Type: Choice
- Required: Yes
- Options: Blocker | Major | Minor | Suggestion

**6. Anything else? (optional)**
- Type: Long text
- Required: No
- Hint: "Errors, logs, screenshots, OS/version"

**7. OK to follow up?**
- Type: Choice
- Required: No
- Options: Yes | No
