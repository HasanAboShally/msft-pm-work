Executive Summary

What This Vision Enables

Scenario 1: Developer in VS Code

Pipeline fails → asks Copilot "Why did this fail?" → agent pulls logs via MCP → identifies memory issue → suggests fix → applies it → re-runs pipeline

4 min vs 20+ min in portal

Scenario 2: Business User in Claude

"Create a Q4 marketing report" → agent discovers semantic models → suggests layout → user refines → creates report via API → schedules refresh

Never opened Fabric portal

Scenario 3: Autonomous CI/CD Agent

PR merged → GitHub Actions triggers → agent deploys to staging → runs validation → tests fail → auto-rollback → posts summary to PR

Zero human intervention

Scenario 4: Visual Pipeline via MCP App

Analytics engineer in Claude → asks to configure pipeline → MCP App renders visual builder inline → user adjusts and confirms → deployed

Portal-like experience, anywhere

Same three layers power all scenarios. The difference is the level of human involvement — configurable, not hard-coded.
